# 🔨 Whack-a-Mole

A fun and interactive browser-based Whack-a-Mole game built with vanilla HTML, CSS, and JavaScript. Moles pop up randomly across a 3×3 grid. Click them before they disappear to score points!

## 🚀 Features

* 🟢 **3×3 grid** of holes with randomly appearing moles
* 🔨 **Custom hammer cursor** that animates on click
* 💥 **Whacked mole animation** where moles change appearance when hit
* 🔊 **Sound effect** plays on every successful hit
* 📊 **Live score tracker** with 10 points awarded per successful hit
* ⚡ **Pure Vanilla JavaScript** with no frameworks or dependencies

## ⚙️ How to Run Locally

### 1. Clone the repository

```bash
git clone https://github.com/kpsreedhar/Whack-a-Mole.git
```

### 2. Navigate into the project folder

```bash
cd Whack-a-Mole
```

### 3. Open the game in your browser

You can simply double-click `index.html` or use a local development server:

```bash
npx live-server
```

> ⚠️ Ensure the `assets/` folder contains all required images and audio files for the game to function correctly.

---

## 🎯 How to Play

1. Open the game in your browser.
2. Moles will randomly pop out of holes on the grid.
3. Move your hammer cursor over a mole and click it before it hides.
4. Each successful hit earns **+10 points**.
5. Try to achieve the highest score possible!
