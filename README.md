# 🔨 Whack-a-Mole

A fun and interactive browser-based Whack-a-Mole game built with vanilla HTML, CSS, and JavaScript. Moles pop up randomly across a 3×3 grid — click them before they disappear to score points!

## 🚀 Features

- 🟢 **3×3 grid** of holes with randomly appearing moles
- 🔨 **Custom hammer cursor** that animates on click
- 💥 **Whacked mole animation** — moles change appearance when hit
- 🔊 **Sound effect** plays on every successful hit
- 📊 **Live score tracker** — earn 10 points per mole
- ⚡ **Pure vanilla JS** — no frameworks or dependencies

## ⚙️ How to Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/whack-a-mole.git
   ```

2. **Navigate into the project folder**
   ```bash
   cd whack-a-mole
   ```

3. **Open `index.html` in your browser**
   ```bash
   # Simply double-click index.html, or use a live server:
   npx live-server
   ```

> ⚠️ Make sure the `assets/` folder is present with all required files for the game to work correctly.

---

## 🎯 How to Play

1. Open the game in your browser
2. Moles will randomly pop out of holes on the grid
3. Move your mouse (hammer cursor) over a mole and **click it** before it hides
4. Each successful hit earns **+10 points**
5. Try to get the highest score possible!
